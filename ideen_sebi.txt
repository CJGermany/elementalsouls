Initialisation:
 - 4 elemente (wasser, feuer, erde, luft)
 - können bekommen werden, indem man sich seelen (perlen?) implantiert
   - implantate können auf einer table ähnlich einem bett gemacht werden
 - sobald man eines der Elemente bekommen hat, wird ein (kleiner) Ladungsbalken irgendwo im HUD eingeblendet
   - zu Anfangs leer, kann aber aufgeladen werden (siehe unten)
 - beim Erlernen eines Elements Fähigkeiten auflisten
 
 
 Skills:
  - Erlernen von Fähigkeiten (basierend auf den Elementen)
  - Erlernen kostet XP (oder?)
  - Fähigkeiten (= Zauber) verbrauchen Ladung (siehe unten)


Ladung:
 - 4 seperat gelistete Ladungen (je Element ein Ladungsbalken im HUD)
 - Rechts und links am Bildschirmrand
 - Werden beim wirken von Zaubern verbraucht
 - Können irgendwie aufgeladen werden und werden zusätzlich mit Rüstung regeneriert (siehen unten)
 

Rüstung: 
 - Setbonus: Wenn man 4 gleichartige Rüstungsteile trägt werden entsprechende Zauber verstärkt
 - Immunität gegen Aura von "soul catcher" Blöcken
 - Bonusregeneration des angezogenen elements
   - NICHT linearer Regenerationsbonus:
   - regenerationRate = (((2^amount)/(amount+1))*amount)*2
   - amount = 1: 2
   - amount = 2: 5.3
   - amount = 3: 12
   - amount = 4: 25.6
   - dadurch erweiterte Spezialisierung auf ein bestimmtes Element


Zauber: (Werden bei gedrückter Alt-Taste + Mausklick erzeugt)
 - Element Wasser
   - WaterWalk: auf dem Wasser laufen (verbraucht wenig, ist nämlich nicht wirklich OP)
   - Dive: Unterwasser besser sehen, schneller fortbewegen, kein Sauerstoffbedarf
   - Waterball: Analog zum Fireball, aber slowt bei Treffer und löscht brennende Objekte und löscht Feuer in 3³ Umkreis
 - Element Feuer
   - FireBall: Entzündet bei Treffer, entzündet Umgebung
   - FireImmunity: Selbsterklärend
 - Element Luft:
   - Flight: ...
   - WindWalk: ...
 - Element Erde:
