initialisation:
 - 4 elemente (wasser, feuer, erde, luft)
 - können bekommen werden, indem man sich seelen (perlen?) implantiert
   - implantate können auf einer table ähnlich einem bett gemacht werden
 - sobald man eines der elemente bekommen hat, wird ein (kleiner) ladungsbalken irgendwo im gui eingeblendet
   - zuanfangs leer, kann aber aufgeladen werden (siehe unten)
 - beim erlernen eines elements fähigkeiten auflisten
 
 skills:
  - erlernen von fähigkeiten (basierend auf den elementen)
  - erlernen kostet XP (oder?)
  - fähigkeiten (= zauber) verbrauchen ladung (siehe unten)

ladung:
 - 4 seperate ladungen (je element ein ladungsbalken im gui)
 - vielleicht rechts unten neben der inventarleiste eingeblendet?
 - werden beim wirken von zaubern verbraucht
 - können irgendwie aufgeladen werden und werden mit rüstung regeneriert (siehen unten)
 

rüstung: 
 - setbonus: wenn man 4 gleichartige rüstungsteile trägt werden entsprechende zauber verstärkt
 - immunität gegen aura von soul catcher blöcken oder so
 - bonusregeneration des angezogenen elements
   - NICHT linearer regenerationsbonus
   - bei einem angezogenen rüstungsteil: +1 regeneration
   - bei zwei angezogenen rüstungsteilen: +5 regeneration
   -  "  drei      "             "      : +15 regeneration
   -  "  vier      "             "      : +30 regeneration
   - dadurch erweiterte spezialisierung auf ein bestimmtes element
